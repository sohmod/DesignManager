
AJAX Example
------------------------------

To run the AJAX example:

  1.  Transfer all of these files into a directory where PHP code can be run
  2.  Open up index.html in a browser.
  
This example uses the following open source modules:

  1.  JSMX (Lightweight Ajax javascript library)    http://www.lalabird.com/
  2.  JSON-PHP  (A library that converts php objects to JSON (javascript) objects
      Found through PEAR (pear.php.net)  http://pear.php.net/pepr/pepr-proposal-show.php?id=198
