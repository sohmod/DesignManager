Shutter - the php image photo gallery

posted by Egg @ 10:42am, Thursday 14 May 2009.

This is a package I created for myself which I thought would be nice to share.

Shutter is an image hosting package which allows users to upload photos in an unlimited number of albums and receive user comments.

In short, it's like flickr but something that YOU have control over.

v4.1.1 Released - Several Fixes and Integrated Langugage Pack

posted by Egg @ 12:09pm, Tuesday 24 February 2009.

eggBlog 4.1.1 has been released and is now available to download.

The language pack has been integrated with the main release (now including SV and FA_IR), system errors are now supressed and the reported $3 link bug has been fixed (tbc).

When updating from pre-v4.0rc3, you will need to manually update your /config.php file to the new language file (from en-uk to en_gb). Then edit your config file through the administration area and save - this will create the neccessary new "tag flag" and "MySQL Database" setting.
